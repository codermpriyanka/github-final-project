# Code of Conduct

## Our Pledge
We pledge to make participation in our project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include:
- Using welcoming and inclusive language
- Being respectful of differing viewpoints
- Gracefully accepting constructive criticism

## Enforcement Responsibilities
Project maintainers are responsible for clarifying standards and taking action.

## Scope
This Code of Conduct applies within all project spaces and public spaces.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Community leaders will follow these guidelines in determining consequences.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
